## What are we building?

We'll build a simple to-do list using the following:

- [htmx](https://htmx.org/)
- [Bun](https://bun.sh/)
- [ElysiaJS](https://elysiajs.com/)
- [Tailwind](https://tailwindcss.com/)
- [Fly.io](https://fly.io/)
